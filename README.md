# JBang Catalog

This is a catalog of aliases to use with [jbang](https://jbang.dev).

To use this you need to have `jbang` installed and available in your PATH.
See the [JBang website](https://jbang.dev/download) for options on how to install JBang.

To see what aliases are available, run the following:

```
$ jbang --fresh catalog list chabala
```
